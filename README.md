
# DevOps Lab

A hands-on DevOps portfolio project focused on core junior-level infrastructure skills. In this lab, I set up an Ubuntu Server virtual machine, installed and managed Nginx, configured basic firewall and SSH security, created a Bash health-check script, automated it with cron, and documented the full workflow in GitHub.

## About the Project

This project was built to practice the foundational skills used in day-to-day DevOps and system administration work. It simulates a small real-world server setup where a service must be installed, verified, secured, monitored, and documented.

The lab covers:

- Linux server administration
- User and group management
- File permissions
- Web server deployment with Nginx
- Firewall configuration with UFW
- SSH service checks
- Log inspection and troubleshooting
- Bash scripting
- Cron-based task automation
- Version control with Git and GitHub

## Objectives

The main goals of this project were to:

- Build confidence working in a Linux server environment
- Learn how to manage and verify services
- Practice reading logs to troubleshoot issues
- Automate a basic operational task with Bash and cron
- Create a clean GitHub repository that documents both the process and the outcome

## Tech Stack

- **OS:** Ubuntu Server
- **Web Server:** Nginx
- **Scripting:** Bash
- **Automation:** Cron
- **Security:** UFW, basic SSH configuration
- **Version Control:** Git
- **Repository Hosting:** GitHub
